Classic AI/ML Research Papers

A Reproduction Portfolio — Reproduced locally on an RTX 2050

This portfolio presents lightweight, functional reproductions of five foundational papers in Artificial Intelligence and Machine Learning. Designed as a "Digital Dean's Study," this project moves beyond standard tutorials to offer rigorous, ground-up implementations in PyTorch. Each model was trained on scaled-down datasets suitable for a personal laptop environment (Intel i5, 16GB RAM, RTX 2050), demonstrating a profound, practical understanding of the underlying architectures and optimization dynamics.

Generative Models

Generative Adversarial Networks

Goodfellow et al., 2014

Implementation of the original GAN architecture. Features a generator and discriminator network locked in a zero-sum game, trained to synthesize handwritten digits from the MNIST dataset.

 NLP / Sequence Modeling

Attention Is All You Need

Vaswani et al., 2017

A complete implementation of the Transformer architecture. Demonstrates scaled dot-product multi-head self-attention by training a character-level generative model on a focused text corpus.

 Computer Vision

Deep Residual Learning for Image Recognition

He et al., 2015

Addressing the vanishing gradient problem through identity skip-connections. Features a custom ResNet-18 model trained from scratch on the CIFAR-10 image classification benchmark.

 Generative Models

Auto-Encoding Variational Bayes

Kingma & Welling, 2013

Exploration of probabilistic generative models. Implements an encoder-decoder architecture that learns a continuous, structured latent space to encode and generate MNIST digits via the reparameterization trick.

 Reinforcement Learning

Playing Atari with Deep Reinforcement Learning

Mnih et al., 2013

A foundational Deep Q-Network (DQN) agent. Demonstrates the integration of deep learning with RL, using experience replay and target networks to solve the classic CartPole environment.
