Attention Is All You Need (Transformer)

Reference: Vaswani et al., 2017

Abstract

This project recreates the foundational Transformer architecture, focusing on the decoder-only generative structure common in modern Large Language Models (LLMs) like GPT. By leveraging multi-head self-attention mechanisms instead of recurrence, the model learns to generate text.

Methodology

I built a miniature character-level Transformer using PyTorch and trained it on the "Tiny Shakespeare" dataset. The architecture includes token and positional embeddings, multi-head causal self-attention blocks, and feed-forward networks.

  • Context Block Size: 8 tokens
  • Embedding Dimension: 32
  • Attention Heads: 4
  • Layers: 3
  • Optimizer: AdamW (lr=1e-3)
