Classic AI/ML Research Papers: A Reproduction Portfolio

Author: Reproduced locally on an RTX 2050

This portfolio contains lightweight reproductions of five highly influential papers in Artificial Intelligence and Machine Learning. The models were implemented in PyTorch and trained on smaller datasets suitable for a personal laptop environment, demonstrating a deep understanding of the core architectures.

Reproduced Papers

  • 1. Generative Adversarial Networks (GANs)

    Goodfellow et al., 2014. Generating handwritten digits using a generator and discriminator network on MNIST.

  • 2. Attention Is All You Need (Transformer)

    Vaswani et al., 2017. A character-level generative model trained on a small text corpus, implementing multi-head self-attention.

  • 3. Deep Residual Learning for Image Recognition (ResNet)

    He et al., 2015. Overcoming the vanishing gradient problem with skip-connections, demonstrated by training a ResNet-18 on CIFAR-10.

  • 4. Auto-Encoding Variational Bayes (VAE)

    Kingma & Welling, 2013. Learning a continuous latent space to encode and generate MNIST digits.

  • 5. Playing Atari with Deep Reinforcement Learning (DQN)

    Mnih et al., 2013. A deep Q-network agent learning to solve the CartPole environment via Q-learning and experience replay.

Deployed via Vercel.
